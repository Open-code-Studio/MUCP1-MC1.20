## 📖 项目描述

这是一个**幽默但充满挑战**的Minecraft 1.20模组包。模组包的目的是通过各种"捣乱"机制让玩家体验一个全新的、充满意外的游戏世界。

**核心特点：**
- 🌪️ **掉落物捣乱** - 物品会不规则地掉落、消失或变化
- 👹 **怪物强化** - 生物更加强大、数量更多
- 🪨 **方块硬度变化** - 挖掘难度突然增加
- 🔧 **合成配方混乱** - 合成表可能会随机变化
- 🌍 **传送门传送混乱** - 不知道会传送到哪里
- 🍃 **重力异常** - 随机的浮空或坠落效果
- ⏰ **时间加速** - 日夜周期快速循环
- 🔊 **音效混乱** - 各种奇怪的音效和声音

## 🚀 快速开始

### 要求
- Minecraft Java版本 1.20（推荐1.20.1）
- Java 17 或更高版本
- 推荐2GB以上内存（4GB最佳）

### 安装步骤

1. **下载Fabric启动器**
   - 前往 [Fabric官网](https://fabricmc.net/)
   - 下载并运行Fabric安装器

2. **克隆或下载本项目**
   ```bash
   git clone https://github.com/Open-code-Studio/MUCP1-MC1.20.git
   ```

3. **复制模组文件**
   - 将 `mods/` 文件夹复制到 `.minecraft/` 目录
   - 将 `config/` 文件夹复制到 `.minecraft/` 目录

4. **启动游戏**
   - 使用Fabric启动器启动Minecraft 1.20

详见 [INSTALL.md](INSTALL.md)

## 🛠️ 配置和定制

2. **禁用特定模组** - 删除或注释掉 `mods/` 中的模组JAR文件
3. **调整难度参数** - 使用数据包命令微调
