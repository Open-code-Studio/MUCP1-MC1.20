# 📋 MUCP1-MC1.20 模组列表

## 🔴 核心"捣乱"模组

### 1. **掉落物捣乱系列**

#### 🎲 Random Drop Time
- **作用**：物品掉落时间随机变化（0.5倍到3倍）
- **难度**：⭐⭐
- **效果**：有时物品瞬间消失，有时物品保留很久
- **提示**：捡起物品时可能就剩0.5秒了！

#### 💨 Quick Despawn
- **作用**：掉落物消失时间加速（默认5分钟变为1分钟）
- **难度**：⭐⭐⭐
- **效果**：必须快速捡起掉落物，否则会丢失
- **提示**：必须保持警惕！

#### 🌀 Item Teleport
- **作用**：掉落物有概率随机传送
- **难度**：⭐⭐⭐⭐
- **效果**：你挖矿得到的物品可能瞬间消失在地球另一端
- **提示**：使用物品管道系统会减少困扰

### 2. **怪物强化系列**

#### 👹 Monster Boost
- **作用**：所有敌对生物伤害提高50%，生命值提高30%
- **难度**：⭐⭐⭐⭐
- **效果**：爬行者爆炸更猛，骷髅箭更快，苦力怕更难躲
- **提示**：必须升级你的盔甲！

#### 🌙 Night Intensifier
- **作用**：夜晚怪物数量增加2倍
- **难度**：⭐⭐⭐
- **效果**：黑夜变成一场真正的战争
- **提示**：夜间离开基地需要做好充足准备

#### ⚡ Mob Speed Up
- **作用**：所有敌对生物移动速度提高40%
- **难度**：⭐⭐⭐⭐⭐
- **效果**：你跑不过苦力怕，骷髅射手比你更快
- **提示**：建立防御工事是必需的

#### 💪 Creeper Enhanced
- **作用**：爬行者爆炸范围扩大，爆炸力增强
- **难度**：⭐⭐⭐⭐⭐
- **效果**：一个爬行者能摧毁更大的区域
- **提示**：远程武器推荐使用

### 3. **方块硬度变化系列**

#### 🪨 Random Block Hardness
- **作用**：方块硬度随机波动
- **难度**：⭐⭐⭐
- **效果**：有时候铁矿秒破，有时候圆石需要挖1分钟
- **提示**：这就是MC的新"魅力"！

#### 🔐 Diamond Lock
- **作用**：钻石矿挖掘时间增加10倍
- **难度**：⭐⭐⭐⭐
- **效果**：找到钻石只是开始，真正的考验才来
- **提示**：选择开采还是放弃？

#### 🌍 Bedrock Breakable

- **难度**：⭐⭐⭐
- **效果**：游戏边界变成虚幻的

### 4. **合成混乱系列**

#### 🎲 Recipe Randomizer
- **作用**：每天0点合成配方随机重新排列
- **难度**：⭐⭐⭐⭐
- **效果**：你之前的合成知识可能完全过时
- **提示**：始终查看合成表（REI推荐）

#### 🔄 Crafting Twist
- **作用**：合成结果有10%概率完全不同
- **难度**：⭐⭐⭐
- **效果**：你想合成剑，但得到了怪异的东西
- **提示**：小心扣动合成按钮！

#### ⚙️ Tool Swap
- **作用**：工具功能有时互换
- **难度**：⭐⭐⭐
- **效果**：斧头变成镐子，锹变成斧头
- **提示**：适应新的"工具配置"

## 🟢 辅助模组和库

这些模组提供支持和优化，但不增加难度：

### 必需模组
- **Fabric API** - Fabric框架基础库
- **Cloth Config API** - 配置管理

### 推荐模组
- **Sodium** - 图形性能优化（FPS提升）
- **Lithium** - 服务器端性能优化
- **Phosphor** - 光照引擎优化
- **Roughly Enough Items (REI)** - 物品和合成表查询（**强烈推荐**在合成混乱模组启用时使用）
- **Mod Menu** - 模组列表和配置界面

### 可选模组
- **Chat Report Fix** - 修复聊天报告问题
- **Better Ping Display** - 更好的延迟显示
- **Controlify** - 游戏手柄支持

## ⚙️ 模组配置

### 难度预设

#### 🟡 简单模式（适合新手）
启用模组：
- Sodium, Lithium, Phosphor（性能）
- REI（合成表）
- 仅启用：掉落物捣乱（轻度）、夜晚怪物增加

#### 🟠 困难模式（标准体验）
启用所有"核心捣乱"模组（第1-8章节）

#### 🔴 地狱模式（终极挑战）
启用所有模组 + 增加强度参数

### 禁用特定模组

不想要某些"捣乱"功能？简单！

1. 打开 `.minecraft/mods/` 文件夹
2. 找到对应的模组JAR文件
3. 删除或将其重命名为`.disabled`后缀

例如：禁用时间加速器
```
RandomTimeJump-1.20.jar → RandomTimeJump-1.20.jar.disabled
```

重启游戏后生效。

## 📞 模组问题反馈

如果某个模组无法工作或产生冲突，请：

1. 检查Minecraft和Java版本是否匹配
2. 删除所有模组配置缓存（`.minecraft/config/`）
3. 重新下载最新版本的模组
4. 在GitHub Issue中报告详细信息
